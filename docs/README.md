# Repository Documentation

This repository provides reusable Terraform modules and complete templates for common AWS use cases.

## Contents
- `docs/NAMING.md`: naming conventions for resources and tags.
- `docs/CONTRIBUTING.md`: contribution workflow and checks.

## Quick Start
1. Pick a template in `templates/`.
2. Copy `terraform.tfvars.example` to `terraform.tfvars`.
3. Update required variables and run `terraform init` then `terraform apply`.
